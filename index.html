📚 三角函数知识点

🌊 振幅 (Amplitude)

定义：振幅是三角函数图像偏离平衡位置的最大距离，决定了函数图像的"高度"。

数学表示：在函数 y = A·sin(x) 中，A 就是振幅

特点：

  • A > 1 时，图像被纵向拉伸
  • 0 < A < 1 时，图像被纵向压缩
  • A < 0 时，图像关于x轴翻转

⚡ 频率 (Frequency)

定义：频率决定了三角函数在单位长度内完成周期性变化的次数。

数学表示：在函数 y = sin(ωx) 中，ω 称为角频率

周期关系：T = 2π/ω（周期与频率成反比）

特点：

  • ω > 1 时，图像被横向压缩，周期变小
  • 0 < ω < 1 时，图像被横向拉伸，周期变大
  • ω 越大，函数变化越快

🔄 相位 (Phase)

定义：相位决定了三角函数图像在水平方向上的平移位置。

数学表示：在函数 y = sin(x + φ) 中，φ 就是初相位

特点：

  • φ > 0 时，图像向左平移 φ 个单位
  • φ < 0 时，图像向右平移 |φ| 个单位
  • 相位决定了函数的起始状态

📊 垂直偏移 (Vertical Shift)

定义：垂直偏移使整个函数图像在垂直方向上平移。

数学表示：在函数 y = sin(x) + k 中，k 就是垂直偏移

特点：

  • k > 0 时，图像向上平移 k 个单位
  • k < 0 时，图像向下平移 |k| 个单位
  • 改变函数的平衡位置

🎯 综合函数形式

一般形式：y = A·sin(ωx + φ) + k

参数说明：

  • A - 振幅，控制图像的纵向伸缩
  • ω - 角频率，控制图像的横向伸缩
  • φ - 初相位，控制图像的水平平移
  • k - 垂直偏移，控制图像的垂直平移

💡 学习建议

通过调节左侧的参数滑块，观察各个参数对函数图像的影响：

  • 先固定其他参数，单独调节一个参数观察变化
  • 使用动画功能观察参数的动态变化效果
  • 尝试预测参数变化后的图像形状，然后验证
  • 注意观察函数性质面板中数值的变化
